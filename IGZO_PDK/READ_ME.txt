The Helvellyn V2.1 PDK consists of the setup and documentation files in the folders "Designkits" and "Documents" respectively. 
The PDK base setup file is "base_Helvellyn_V2.1.0.beta.2.hotfix.1.tar.gz".
The newly released patch file "patch_Helvellyn 2.1.0.beta.2hotfix.3.tar.gz" has Spice model changes and updates according to the latest changes in the fabrication process. See "Helvellyn 2.1.0.beta.2.hotfix.3 Release Contents.pdf" for full details.

Four installation files in FlexIC / Designkits:
	- base_Helvellyn_V2.1.0.beta.2.hotfix.1.md5
	- base_Helvellyn_V2.1.0.beta.2.hotfix.1.tar.gz
	- patch_Helvellyn_V2.1.0.beta.2.hotfix.3.md5
	- patch_Helvellyn_V2.1.0.beta.2.hotfix.3.tar.gz

Seven document files in FlexIC / Documents:
	- Helvellyn 2.1.0.beta.2 Datasheet.pdf
	- Helvellyn 2.1.0.beta.2 Design Guide.pdf
	- Helvellyn 2.1.0.beta.2 Design Rules Manual.pdf
	- Helvellyn 2.1.0.beta.2 Reticle Design Manual.pdf
	- Helvellyn 2.1.0.beta.2.hotfix.1 Installation Guide.pdf
	- Helvellyn 2.1.0.beta.2.hotfix.1 Release Contents.pdf
	- Helvellyn 2.1.0.beta.2.hotfix.3 Release Contents.pdf

FOR FIRST TIME INSTALLATION OF PDK:
	- Use the instructions in the document "Helvellyn 2.1.0.beta.2.hotfix.1 Installation Guide.pdf"
	- Apply Hotfix 3 patch on the installation. (Follow next step) 

FOR APPLYING HOTFIX 3 PATCH ON EXISTING PDK INSTALLATION:
	- Unpack "patch_Helvellyn_V2.1.0.beta.2.hotfix.3" folder. The installation should overwrite 1 file (nch.scs) in the previously installed PDK structure (pdk/models/nch.scs).
